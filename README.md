# PharmaCore

A modern pharmaceutical company website built with Next.js 16, React 19, and Tailwind CSS v4.

🌐 **Live Demo:** [https://pharmacore-iota.vercel.app](https://pharmacore-iota.vercel.app)

---

## Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org)
- **UI:** [React 19](https://react.dev) + [Tailwind CSS v4](https://tailwindcss.com)
- **Icons:** [Lucide React](https://lucide.dev)
- **Language:** TypeScript
- **Deployment:** [Vercel](https://vercel.com)

## Pages

- `/` — Home
- `/about` — About Us
- `/products` — Products
- `/research` — Research
- `/quality` — Quality
- `/blog` — Blog
- `/careers` — Careers
- `/contact` — Contact

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run start` | Start production server |
| `npm run lint` | Run ESLint |

## Deployment

Deployed on [Vercel](https://vercel.com). Every push to `main` triggers an automatic deployment.
